Project Overview : 
This project presents an E-Commerce Sales Dashboard built in Microsoft Excel to analyze sales performance, customer behavior, product category trends and payment preferences. The dashboard transforms raw transactional data into actionable insights through interactive visualizations and KPI tracking.

Objectives :
The dashboard was designed to:
- Monitor overall sales performance and profitability.
- Analyze customer distribution across different age groups and locations.
- Identify top performing product categories.
- Evaluate the impact of discount campaigns on sales.
- Understand customer payment method preferences.
- Support data driven business decision making.

Dashboard Features :
- Key Performance Indicators (KPIs)
- Net Profit: 158,177,240
- Number of Customers: 55,000
Interactive Filters (Slicers) :
- Age Bracket
- Discount Name
Visualizations :
- Net Income Per Year
- Customer Location Analysis
- Net Income Per Product Category
- Net Income Per Payment Method

Tools & Techniques Used :
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design & Formatting 
- Data Cleaning and Transformation
- KPI Reporting

Key Insights :
- Profitability trends can be monitored across multiple years.
- Electronics generated the highest net income among product categories.
- Credit Card and Debit Card payments contributed the largest share of revenue.
- Customer purchasing behavior varies across different locations and age groups.
- Discount campaigns can be filtered and analyzed to evaluate their effectiveness.
